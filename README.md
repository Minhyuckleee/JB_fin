# JB Fin AI Agent Reference

이 브랜치는 대회 요강, 아이디어 원문, 참고자료, 리서치 자료를 보관합니다.

이 브랜치의 자료는 Agent runtime에 직접 포함되지 않습니다.  
개발에 반영할 내용은 요약/정제 후 `develop` 브랜치의 `docs/` 또는 `data/processed/`에 반영합니다.

## Structure

- `rules/`: 대회 요강 및 제출 양식
- `idea/`: 초기 아이디어 문서
- `research/`: 외부 리서치 자료
- `raw_sources/`: 원본 자료
