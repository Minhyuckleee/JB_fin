- 서비스 개요(Service)

- 핵심 기능 명세(Feature Specification)

## 10. 주요 제약사항

- 반드시 제시된 주제 중 1개의 주제만 선택해야 함
- 선택한 주제 범위 내에서는 세부 서비스 아이디어, 구현 방식, 적용 시나리오를 자유롭게 설계할 수 있음
- AI Agent 서비스는 판단, 행동, 검증/개선 요소를 포함해야 함
- 개발에 필요한 컴퓨팅 리소스는 별도로 제공되지 않음
- 오프라인 본선에서도 참가자 본인의 장비로 개발 및 시연 가능해야 함
- 예선 제출 자료는 PDF 형식이어야 함
- 시연영상은 선택 제출물이지만 제출 시 5분 이내, 100MB 제한
- 시연영상은 대회명, 팀명, 서비스명 등으로 외부 검색 시 노출될 수 있는 공개 영상 플랫폼에 업로드해서는 안 됨
